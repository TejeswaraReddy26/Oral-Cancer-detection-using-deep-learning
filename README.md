## 📌 Overview
Early detection of oral cancer is crucial for improving survival rates. This project presents a deep learning-based system that automatically detects oral cancer from medical images using Convolutional Neural Networks (CNNs).

The model analyzes oral cavity images and classifies them as **cancerous** or **non-cancerous**, assisting healthcare professionals in faster and more reliable diagnosis.

---

## 🚀 Features
- Automated oral cancer detection using CNN  
- Image preprocessing with OpenCV and PIL  
- Accurate classification of medical images  
- Reduces manual diagnostic effort  
- Scalable for real-world healthcare applications  

---

## 🛠️ Tech Stack
- Python  
- TensorFlow / Keras  
- OpenCV  
- NumPy, Pandas  

---

## 🔄 Workflow
1. Data Collection (oral image dataset)  
2. Image Preprocessing (resize, normalization)  
3. Model Training (CNN architecture)  
4. Evaluation (accuracy, loss metrics)  
5. Prediction (new image classification)  

---

## 📊 Results
The model effectively distinguishes between cancerous and non-cancerous images, supporting early detection and improving diagnostic efficiency.
